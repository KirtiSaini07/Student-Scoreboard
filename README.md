# Student Scoreboard

## 📌Project Description
Create a Student Scoreboard Application that allows users to:
- View a list of students and their scores
- Update student scores dynamically
- Add new students using a form
- Display pass/fail status based on marks

---

## 📌Tools & Technologies
- React (with Vite)
- JavaScript (JSX)
- CSS (Pure CSS only, no Tailwind)

---

## 📌Features
- Add students with their scores
- View all students in a clean table
- Real-time score updates
- Simple and intuitive UI
